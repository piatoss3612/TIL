 # 01. Solana-Cli 설치하기

- 참고: https://docs.solana.com/cli/install-solana-cli-tools

 ## 1. 설치

```bash
$ sh -c "$(curl -sSfL https://release.solana.com/v1.16.13/install)"
```

## 2. 설치 확인

```bash
$ solana --version
solana-cli 1.16.13 (src:b2a38f65; feat:3949673676, client:SolanaLabs)
```

## 3. 업데이트 확인

```bash
$ solana-install update
```

## 4. 도움말

```bash
$ solana --help
```

## 5. 특정 명령어 도움말

```bash
$ solana [명령어] --help
```
